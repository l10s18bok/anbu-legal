1. About the Service

Anbu is a daily wellness check application that helps families quietly monitor the well-being of aging parents and loved ones living alone. The Service operates by sending automatic daily wellness signals ("heartbeats") from the monitored person's device to connected family members.

  1. "Service" means the Anbu mobile application and all related features provided by us.

  4. "Heartbeat" means the daily wellness signal automatically transmitted from a subject's device.
  5. "Invite Code" means the unique code issued to a subject, used to establish a connection with a guardian.

4. Service Description

4.1 Subject Features (Free)

  • Automatic daily heartbeat transmission at a scheduled time
  • Manual wellness report ("I'm okay") button

4.2 Guardian Features (Subscription)
